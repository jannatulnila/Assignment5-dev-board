Question-1: What are the different ways to select an element in the DOM?

elements can be selected in different ways using javaScript. For example getElementByID(), getElementByClassName(), getElementByTagName(), querySelector(), querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerHTML property recognizes HTML tags and renders the content according to the tags. innerText and textContent ignore HTML tags and treat them as part of the text

Question-3: What is event delegation in the DOM?

Event delegation is a JavaScript technique that optimizes event handling by attaching a single event listener to a parent element, rather than multiple listeners to child elements

Question-4: What is event bubbling in the DOM?

Event bubbling can be defined as a phase of Event propagation in which if an event occurs on a particular element will propagate or bubble up to the ancestor or the parent elements in the DOM hierarchy

Question-5: How do you create, add, and remove elements using JavaScript?

For create an element we can use document.createElement. After create an element we can add it to the page using methods like: append(), appendChild(), innerHTML. we can remove that element using js: remove(), removeChild()
